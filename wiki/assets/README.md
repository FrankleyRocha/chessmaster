# Assets - Diagramas

Esta pasta contém diagramas PNG do vault ChessLM.

## Nota sobre Diagramas

Todos os diagramas complexos foram convertidos para ASCII art para garantir compatibilidade máxima com diferentes renderizadores de Obsidian.

Os diagramas mermaid simples (graph LR, graph TD) sem caracteres especiais foram mantidos.

## Diagramas Convertidos

Os seguintes tipos de diagramas foram convertidos para ASCII:

1. **timeline** - Não suportado na maioria das versões do Obsidian
2. **pie** - Gráficos de pizza com suporte inconsistente
3. **subgraph com caracteres especiais** - Labels com `<br/>`, `✓`, etc.
4. **Nós com vírgulas/listas** - Conteúdo como `["a", "b", "c"]`

## Verificação

Os diagramas mermaid restantes foram testados e funcionam corretamente no Obsidian.
